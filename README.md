# IP Blocklist App

This project is a simple IP Blocklist application built with React, TypeScript and TailwindCSS.

## 🚀 Features

- Add IP address
- Validate IP format using Regex
- Prevent duplicate IP entries
- Delete IP from list
- Persist data with localStorage

## 🛠️ Technologies

- React (Vite)
- TypeScript (TSX)
- TailwindCSS
- useState & useEffect

## 📦 Installation

```bash
npm install
npm run dev
```

## 🧠 What I Learned
- State management with useState
- Handling side effects with useEffect
- Controlled components in React
- Input validation with Regex
- Working with localStorage
- Building reusable UI components

## 🔒 Security Perspective
This project simulates a basic IP blocklist interface, which is commonly used in cybersecurity tools such as firewalls and SOC dashboards.
